Glasswall ICAP Adaptation Service on Compliant Kubernetes#

Added features:

  • Ingress controller (Nginx)
  • Policy enforcement (OPA)
  • Log collector/forwarder (Fluentd)
  • Metrics collection and monitoring (Prometheus/Grafana)
  • Intrusion detection and breach reporting (Falco)
  • Certificate management (Cert-manager)
  • Centralized logging Service (Elasticsearch/Kibana)

To be added features:

  • Metrics collection and monitoring (Prometheus/Grafana)
    • Activate alerting using Prometheus AlertManger
    • Custom grafana dashboards
  • Backup/restore Service (Velero)
    • Only for Service cluster resources as workload cluster are single node disposable clusters
  • Elasticsearch and Influxdb backup to s3 bucket as part of disaster recovery.
  • Container registry, including vulnerability scanning (Harbor)
  • Single sign-on Service (Dex)
  • Regulate use of credentials.
